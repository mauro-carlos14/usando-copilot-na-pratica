# usando-copilot-na-pratica
## Sobre

Este repositório contém código de teste do GitHub Copilot no Visual Studio Code.

## Objetivo

- Aprender a usar Copilot para gerar e completar código.
- Testar integração com o VS Code.
- Validar sugestões e workflows de desenvolvimento.

## Como usar

1. Abra o projeto no Visual Studio Code.
2. Ative o GitHub Copilot (extensão instalada/ativada).
3. Edite arquivos e aceite sugestões.
4. Teste a execução do código localmente.

## Estrutura

- `README.md`: documento principal.
- Outros arquivos de teste criados durante o experimento.

## Observações

- Foco em aprendizado e prática.
- Não é código de produção.
